axum

cookie 
    ทำให้ใช่งานความสามารถจาก cookie function

    ทำให้ trait รองรับ async function ได้

    ใช้สำหรับการทำให้เราเขียน concurrency ในรูปแบบของ async ได้ 
tower, tower-http
    ใช้จัดการกับ middleware
tracing, tracing-subscriber 
    ใช้ทำ logging
serde, serde_json 
    ใช้สำหรับการจัดการ JSON Format
chrono 
    ใช้จัดการเรื่อง format เวลา
diesel

dotenv

anyhow

jsonwebtoken
